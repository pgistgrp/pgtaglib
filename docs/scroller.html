PGIST Tag - scroller
Add the following line to the beginning of your jsp file:
    <%@ taglib uri="http://www.pgist.org/jsf/components" prefix="pg" %>


Backing Bean:

    public class BackingBean() {

        private PageSetting pageSetting;
        public PageSetting getPageSetting() {
            return pageSetting;
        }

        public String listUser(ActionEvent event) {
            ......
        }

    }//class BackingBean()


Tag Usage Paradigm:

<pg:scroller id="scroller" width="80%"
             actionListener="#{SysAdminBean.listUser}"
             value="#{SysAdminBean.pageSetting}"
             showPageGo="true"
             showRowsOfPage="true"
             showPageNumber="10"
             infoType="row" />


Options:

    id : required, a unique name for current scroller
    width: optional, default is 100%
    actionListener: required, should be a method of the backing bean
    value: required, should be an instance of org.pgist.util.PageSetting
    showPageGo: optional, default is not showing the directly page going component
                valid value: true
    showRowsOfPage: optional, default is not showing the rows of page component
    showPageNumber: optional, default is showing 10 pages
                valid value: positive numbers, default is 10
    infoType: optional, default is not showing the information
              valid value: row, showing current row range and total row bumber
                           page, showing current page and total page number
